# 12.如果new了之后出了问题直接return。会导致内存泄漏。怎么办（智能指针，raii）

1.检查。在return的时候，**判断指针是否为空**，如果为[空指针](https://so.csdn.net/so/search?q=空指针&spm=1001.2101.3001.7020)，再进行return。

2.可以使用[智能指针](https://so.csdn.net/so/search?q=智能指针&spm=1001.2101.3001.7020)。
