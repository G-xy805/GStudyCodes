# 20.介绍一下const

1、const是一个C语言（ANSI C）的关键字，具有着举足轻重的地位。

2、它限定一个变量不允许被改变，产生静态作用。

3、使用const在一定程度上可以提高程序的安全性和可靠性。

4、另外，在观看别人代码的时候，清晰理解const所起的作用，对理解对方的程序也有一定帮助。

5、另外CONST在其它编程语言中也有出现，例如Pascal、C++、PHP5、B#.net、HC08 C、C#等。
